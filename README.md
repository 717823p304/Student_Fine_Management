STUDENT FINE MANAGEMENT SYSTEM

A simple Java-based console application to manage student fines efficiently within an institution.

This project is developed using Core Java and follows an object-oriented approach to handle users, students, payments, and fine management.

FEATURES

1. User Login System
2. Admin Module
3. Cashier Module
4. Student Fine Tracking
5. Fine Payment Management
6. Fine Records Maintenance

TECHNOLOGIES USED

1. Java (Core Java)
2. OOP Concepts
3. Eclipse IDE
4. Git & GitHub
